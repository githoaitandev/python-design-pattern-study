# 02. Machine Learning Basics

Topic này giải thích cách model học từ dữ liệu.

Bạn không cần giỏi toán ngay từ đầu, nhưng cần hiểu các ý tưởng như `Training`, `Dataset`, `Label`, `Loss`, `Overfitting`, và `Generalization`.

## Bạn sẽ học

- Model học bằng cách giảm sai số.
- Dữ liệu tốt quan trọng như thuật toán tốt.
- Model giỏi trên dữ liệu huấn luyện chưa chắc giỏi ngoài đời.
