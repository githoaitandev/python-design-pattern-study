# Glossary

## Dataset

`Dataset` là tập dữ liệu dùng để huấn luyện hoặc đánh giá model.

## Training

`Training` là quá trình model học từ dữ liệu bằng cách điều chỉnh các tham số bên trong.

## Label

`Label` là đáp án đúng trong dữ liệu huấn luyện có giám sát. Ví dụ ảnh con mèo có label là “cat”.

## Feature

`Feature` là thông tin đầu vào mà model dùng để học. Trong dữ liệu khách hàng, feature có thể là tuổi, số lần mua hàng, hoặc khu vực.

## Loss Function

`Loss Function` là hàm đo model sai bao nhiêu. Loss càng thấp thường nghĩa là model dự đoán càng gần đáp án.

## Overfitting

`Overfitting` là khi model học quá sát dữ liệu huấn luyện, đến mức kém trên dữ liệu mới.

## Generalization

`Generalization` là khả năng hoạt động tốt trên dữ liệu chưa từng thấy.
