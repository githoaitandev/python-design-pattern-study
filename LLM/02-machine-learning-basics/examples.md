# Examples

## Dự đoán giá nhà

Input có thể gồm diện tích, vị trí, số phòng. Label là giá nhà thật. Model học mối quan hệ giữa input và giá.

## Phân loại cảm xúc

Input là câu đánh giá của khách hàng. Label có thể là `positive`, `neutral`, hoặc `negative`.

## Overfitting trong thực tế

Một model đạt 99% trên dữ liệu huấn luyện nhưng chỉ 60% trên dữ liệu mới có thể đã overfit.
