# Mental Model

LLM không đọc như con người đọc từng “ý” hoàn chỉnh. Nó xử lý chuỗi token và dự đoán token tiếp theo dựa trên ngữ cảnh.

Điều quan trọng: LLM không tự động biết đâu là sự thật mới nhất. Nó sinh câu trả lời dựa trên mẫu đã học và ngữ cảnh bạn cung cấp. Vì vậy, nếu cần thông tin chính xác, mới, hoặc thuộc tài liệu riêng, bạn phải cung cấp nguồn hoặc dùng retrieval.
