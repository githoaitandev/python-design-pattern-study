# Glossary

## Large Language Model

`Large Language Model` hay `LLM` là model ngôn ngữ lớn được huấn luyện trên lượng dữ liệu rất lớn để xử lý và sinh ngôn ngữ.

## Token

`Token` là đơn vị văn bản mà model xử lý. Nó có thể là một từ, một phần của từ, dấu câu, hoặc ký hiệu.

## Context Window

`Context Window` là lượng token tối đa model có thể nhìn thấy trong một lần xử lý.

## Prompt

`Prompt` là input bạn đưa cho model.

## Completion

`Completion` là output model sinh ra.

## System Prompt

`System Prompt` là chỉ dẫn cấp cao định hướng cách model nên hành xử.

## Hallucination

`Hallucination` là khi model tạo thông tin sai hoặc không có căn cứ nhưng nghe có vẻ tự tin.
