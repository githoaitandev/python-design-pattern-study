# Glossary

## Prompt Engineering

`Prompt Engineering` là cách thiết kế input để model tạo output hữu ích hơn.

## Zero-shot Prompting

`Zero-shot Prompting` là yêu cầu model làm việc mà không đưa ví dụ.

## Few-shot Prompting

`Few-shot Prompting` là đưa vài ví dụ mẫu để model bắt chước định dạng hoặc cách suy nghĩ.

## Temperature

`Temperature` điều chỉnh độ ngẫu nhiên. Thấp thì ổn định hơn, cao thì đa dạng hơn.

## Top-p

`Top-p` giới hạn nhóm token có xác suất cao mà model được chọn.

## Max Tokens

`Max Tokens` giới hạn độ dài output.

## Structured Output

`Structured Output` là yêu cầu model trả về theo cấu trúc rõ ràng như JSON.
