# Mental Model

LLM bình thường giống người tư vấn. LLM có tool giống người tư vấn có thể mở máy tính, tra dữ liệu, chạy lệnh, hoặc gọi API.

Agent giống một cộng sự có thể làm việc nhiều bước, nhưng vẫn cần ranh giới rõ:

- Được dùng công cụ nào?
- Khi nào phải hỏi người dùng?
- Có được ghi dữ liệu không?
- Nếu tool lỗi thì làm gì?
