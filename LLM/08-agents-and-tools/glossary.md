# Glossary

## Tool Calling

`Tool Calling` là khả năng model yêu cầu gọi một công cụ bên ngoài, ví dụ API thời tiết, database, hoặc hàm tính toán.

## Function Calling

`Function Calling` là một dạng tool calling, nơi model tạo tham số để gọi một function đã định nghĩa.

## Agent

`Agent` là hệ thống dùng LLM để lập kế hoạch, dùng công cụ, quan sát kết quả, và tiếp tục hành động qua nhiều bước.

## Workflow

`Workflow` là chuỗi bước được thiết kế sẵn. Workflow thường dễ kiểm soát hơn agent tự do.

## Planning

`Planning` là quá trình chia mục tiêu lớn thành các bước nhỏ.

## Human-in-the-loop

`Human-in-the-loop` là thiết kế có con người kiểm tra hoặc phê duyệt ở các bước quan trọng.
