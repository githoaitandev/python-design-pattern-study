# Examples

## Tool calling đơn giản

Người dùng hỏi thời tiết. Model không tự đoán, mà gọi tool thời tiết rồi tóm tắt kết quả.

## Agent coding

Agent đọc code, chạy test, sửa file, chạy test lại, rồi báo kết quả.

## Workflow có kiểm soát

Một hệ thống tạo báo cáo có thể chạy theo workflow cố định: lấy dữ liệu, phân tích, tạo bảng, viết tóm tắt, gửi người duyệt.
