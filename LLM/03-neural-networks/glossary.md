# Glossary

## Neural Network

`Neural Network` là mô hình gồm nhiều lớp tính toán nối với nhau để biến input thành output.

## Layer

`Layer` là một tầng xử lý trong neural network.

## Parameter

`Parameter` là giá trị bên trong model được học trong quá trình training.

## Weight

`Weight` là một loại parameter quyết định mức ảnh hưởng của tín hiệu đầu vào.

## Activation Function

`Activation Function` giúp neural network học quan hệ phi tuyến, tức là các mẫu không đơn giản theo đường thẳng.

## Gradient Descent

`Gradient Descent` là cách điều chỉnh parameter từng chút một để giảm loss.
